Dashboard guide

This is a functional admin prototype. All data is persisted in the browser's localStorage. That means:

  • Gallery reorders and removals stick across refreshes on this device.
  • Inquiries submitted from the contact form show up here.
  • Nothing is synced to a server. Clearing browser data resets the state.

Moving to production

Swap three things to make this real:

  1. Auth. Replace the demo check in assets/js/admin.js with Firebase Auth, Supabase, or a Node+Express session.
  2. Gallery storage. Replace localStorage with a real DB (Supabase/Firestore). Uploads go to S3 or Cloudinary.
  3. Inquiries. POST the contact form to Formspree, Resend, or an API route that sends an email to mtframephotography@gmail.com.
